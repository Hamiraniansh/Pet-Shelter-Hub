A modern, responsive web app for a pet shelter. Users can browse pets, learn about adoption, and connect with shelters to give animals loving homes.

Features:

Browse adoptable pets

Pet profiles with images & details

Adoption inquiry/contact form

Responsive design for mobile & desktop

Tech Stack:

HTML, CSS, JavaScript

Optional: Backend (Node.js / PHP / Firebase)

How to Run:

Clone the repository:

git clone https://github.com/mubin25s/pet-shelter.git

Open index.html in a browser.

Optional: Set up a backend for adoption submissions.
